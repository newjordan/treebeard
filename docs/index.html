Reproduction contract

The quality score uses one model slot and one benchmark worker. It is distinct from saturated 12-worker throughput testing.

Serving262,144 total context, one fixed slot, unified KV, F16 KV, flash attention.
Evaluation69 scenarios, one worker, 180-second timeout, temperature 0, thinking disabled.
ControlPinned model checksum, pinned runtime, pinned benchmark commit, seed 42.
