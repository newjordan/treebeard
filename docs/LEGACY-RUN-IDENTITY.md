# Immutable run identity note

The 94/100 benchmark completed before the project was renamed Treebeard. Its
raw JSON, logs, and restored-service properties therefore contain the previous
model alias and host name. Those fields are preserved byte-for-byte so the
published SHA-256 and audit trail remain valid.

Treebeard is the current and only project name. The legacy strings in this
evidence directory are historical run identifiers, not current branding.
